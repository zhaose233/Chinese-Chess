# Chinese Chess

#### 介绍
用SDL2实现的中国象棋

#### 软件架构
支持SDL2的任何架构


#### 安装教程

1.  git clone https://github.com/weiliang1503/chinese-chess.git
2.  cd chinese-chess
3.  gcc chess_sdl2.c \`sdl2-config --cflags --libs\` -lSDL2_ttf -lSDL2_image -o chess_sdl2
4.  ./chess_sdl2

#### 使用说明

1.  依赖SDL2
2.  使用gcc构建
